Install
-------------------------------------------------------------------------------

``elementary_math`` is released on PyPI, so all you need is:

.. code-block:: console

	$ pip install elementary_math

To upgrade to latest version:

.. code-block:: console
	
	$ pip install --upgrade elementary_math
